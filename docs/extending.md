# Extending Open Truly Chat

You can easily customize Open Truly Chat or add new features:

## Add New UI Features
- Create new React components in `/client/src` for the dashboard or chat uploads.

## Change Bot Logic
- Edit `index.js` or `gpt.js` to change how the bot replies or add new backend features.

## Custom Styling
- Update `/client/src/App.css` and `/client/src/index.css` for your own look and feel.

## Example Ideas
- Add support for more chat platforms
- Improve reply logic with new AI models
- Add notifications or analytics

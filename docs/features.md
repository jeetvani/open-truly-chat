# Features of Open Truly Chat

## Main Features

- **Learns your style:** Bot replies just like you, using your WhatsApp chat history.
- **Private chats only:** Only replies in 1:1 chats, not groups or status.
- **Web dashboard:** Easy interface to upload chats, set API key, and monitor bot activity.
- **Secure WhatsApp linking:** Scan QR code to connect your WhatsApp safely.
- **Safe uploads:** Only .txt files, up to 10MB, filenames sanitized.
- **Real-time feedback:** See QR code, connection status, and incoming messages instantly.
- **Handles errors gracefully:** If something fails (like missing API key), bot responds politely.
